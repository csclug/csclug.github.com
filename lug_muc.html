How To Connect to the Conference Chat Server

From Hacking to Tunneling to Clouds and Green Pastures, learn from our fun group.

This is a howto describing connecting to the chat conference room. Anyone can connect and just "hang out" - or if you have specific questions or ideas this is a great way to communicate those. Even if no one is in the room, feel free to type a message and someone will eventually respond (you can see the history for the room in your client as well as online).

Install a Jabber client

You will first need to download and install a Jabber client. A list of clients can be found at http://xmpp.org/xmpp-software/clients/. A good one is Pidgin, which runs on Linux and Windows. The following instructions assume that you're using Pidgin.

Create an account

Jabber is different from other chat protocols like AOL Instant Messanger and MSN Messanger which have a single server. There are literally hundreds of Jabber servers. You can create an account on any of the public Jabber servers. Each Jabber server can talk to any of the other Jabber servers, enabling you to then talk to people no matter which Jabber server they are using. This means that it really doesn't matter which server you create an account on. If you have a GMail account, you can connect to Google's jabber server by following the instructions here (then skip to the "Join Chat Room" section).

To create an account, first start Pidgin. Then click on the "Accounts" button. Click "Add", and then "Jabber" from the top dropdown. Type in your preferred screen name. For the "server" field, either use jabber.org or one of the other servershere. Your resource is just a description of the computer you are using, and can be anything (like "home computer" or "my laptop"). Then type in your preferred password, and click "Register".

Join a chat room

Once you are signed in to a Jabber server, click "Buddies" at the top, then "Add Chat". The room is "csclug" and the server is "conference.butterfat.net". You don't need a password. Just click "Add". The room "csclug" should have been added to your buddy list. Just double click on the room, and you should be in the room.
